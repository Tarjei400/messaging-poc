SSE cross-instance cluster

The broker is a horizontal-scaling backplane: a message published from any instance reaches a client connected to a different instance — no presence registry, just topic routing.

Demo: 3 users — alice & bob in org acme, carol in org globex. Suggested wiring: alice on :7001, bob on :7002, carol on :7001.
  • Send to user alice → only alice.
  • Send to org acme → alice + bob, across both instances.
  • Send to org globex → only carol.
